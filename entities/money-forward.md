# Money Forward

## 它是什么
Money Forward 是这篇 Cursor 客户案例中的采用方。它的价值不在于“又一个用了 AI 的公司”，而在于它提供了一个跨 engineering、QA、product、design 的真实组织级 adoption 案例，说明 coding agents 开始从工程团队扩展到整个软件交付链条。

## 这篇案例里的关键信号
- 超过 1,000 名员工 daily use Cursor。
- 工程团队中，开发者个人每周节省 15–20 小时。
- QA 团队测试用例生成速度提升 70%。
- Product 开始直接读取 production code 来补强 PRD 和规格。
- Design 开始直接基于 live frontend 与产品分析数据迭代，而不是只看静态稿。

## 为什么它值得进库
Money Forward 这篇案例把“coding agents 只对工程师有用”的假设打破了。它展示的是一种 cross-functional adoption 路线：
- 先在 engineering 里证明价值
- 再向 QA、product、design 扩展
- 最后把 agent 变成贯穿 SDLC 的通用工作台
