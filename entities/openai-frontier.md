# OpenAI Frontier

## 它是什么
OpenAI Frontier 是 OpenAI 面向企业安全部署 agents 的平台层叙事。在当前知识库里，它的价值不只是“另一个企业产品页”，而是它为 Ryan Lopopolo 在 Latent.Space 文章里描述的 extreme harness engineering / dark factory 实验提供了更正式的平台上下文。

## 当前可确认的信号
- OpenAI 把 Frontier 定位为 enterprise AI transformation 的平台，而不是单一模型 API。
- 文章与访谈共同指向：Frontier 关心的不是单次 agent 调用，而是如何把 agents 安全、可治理、可观察地部署进企业真实流程。
- Latent.Space 的文章把 Frontier 团队描述成 OpenAI 内部最激进的 Codex 用户：
  - >1M LOC
  - >1B tokens/day
  - 零人工写码
  - 接近零人工 pre-merge review
- 这说明 Frontier 在知识库里的位置，更像“企业级 agent runtime / governance / transformation 平台”，而不是普通 coding assistant。

## 它在知识库中的位置
[[openai-frontier]] 让 [[openai]] 这条线从 “harness engineering 文章” 进一步走到 “平台与组织运行面” 这一层。它和 [[cloud-agent-operations]]、[[symphony]]、[[entangled-software]] 都有关，但重点不同：
- Frontier 更像平台母体
- Symphony 更像其中一条 orchestration / work execution 路线
